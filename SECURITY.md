# Security Policy

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability
Please report security vulnerabilities via the issue tracker or directly to the maintainers. Do not open public issues for security problems.

## Best Practices
- Do not commit secrets or credentials.
- Keep dependencies up to date.
- Review third-party code before inclusion.
